1. git branch <branch-name>
2. git checkout <branch-name>
3. git add . & git commit -m "commit message"
4. git push --set-upstream origin <branch-name>
5. git merge <branch-name> (dont forget to check out to make branch before typing this command)
6. Get back to the respsitory -> create a pull request -> merge the pull request
7. git branch -d <branch-name>
git clone: Creates a local copy of a remote.
  git remote -v: Lists a Git project’s remotes.
  git fetch: Fetches work from the remote into the local copy.
  git merge origin/master: Merges origin/master into your local branch.
  git push origin <branch_name>: Pushes a local branch to the origin remote.
